# Fit Training - PWA

App PWA para vincular personal trainers aos seus alunos.

## Instalação

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor:
```bash
npm start
```

3. Acesse no navegador:
```
http://localhost:3000
```

## Funcionalidades

- **Tela Inicial**: Logo animada com halter
- **Cadastro/Login**: Sistema de autenticação
- **App do Aluno**:
  - Gráfico de progresso
  - Instruções do treinador
  - Rotina semanal de treino
  - Chat com o treinador
  - Registro de peso e medições
- **Painel Admin**: Gerenciar alunos e ver resultados

## Tecnologias

- HTML/CSS/JavaScript
- Node.js + Express
- PWA (Service Worker + Manifest)
